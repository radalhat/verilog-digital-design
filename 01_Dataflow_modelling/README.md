# Dataflow Modelling

## Overview

This folder contains digital circuits implemented using the **Dataflow Modelling** style in Verilog HDL. Each project includes the Verilog design, a dedicated testbench for functional verification, and supporting documentation.

## Folder Structure

```
01_Dataflow_Modelling/
│
├── 01_Logic_Gates/
├── 02_Combinational_Circuits/
└── 03_Sequential_Circuits/
```

Each circuit folder contains:

- Verilog HDL design (`.v`)
- Testbench (`*_tb.v`)
- Circuit-specific `README.md`
- Logic diagram (where applicable)
- Simulation waveform (where applicable)
