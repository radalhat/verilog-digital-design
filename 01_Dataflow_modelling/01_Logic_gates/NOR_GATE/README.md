# NOR Gate

## Description

2-input NOR gate implemented using **Dataflow Modelling** in Verilog HDL.

## Files

- `nor_gate.v` – Design
- `nor_gate_tb.v` – Testbench

## Key Concepts

- Dataflow Modelling
- Continuous Assignment (`assign`)
- Boolean NOR Operation
- Functional Verification

## Output

`Y = ~(A | B)`
