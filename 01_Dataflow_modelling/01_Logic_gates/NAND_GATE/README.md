# NAND Gate

## Description

2-input NAND gate implemented using **Dataflow Modelling** in Verilog HDL.

## Files

- `nand_gate.v` – Design
- `nand_gate_tb.v` – Testbench

## Key Concepts

- Dataflow Modelling
- Continuous Assignment (`assign`)
- Boolean NAND Operation
- Functional Verification

## Output

`Y = ~(A & B)`
