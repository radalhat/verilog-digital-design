# OR Gate

## Description

2-input OR gate implemented using **Dataflow Modelling** in Verilog HDL.

## Files

- `or_gate.v` – Design
- `or_gate_tb.v` – Testbench

## Key Concepts

- Dataflow Modelling
- Continuous Assignment (`assign`)
- Boolean OR Operation
- Functional Verification

## Output

`Y = A | B`
