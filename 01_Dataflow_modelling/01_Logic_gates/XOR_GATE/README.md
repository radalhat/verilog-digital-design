# XOR Gate

## Description

2-input XOR gate implemented using **Dataflow Modelling** in Verilog HDL.

## Files

- `xor_gate.v` – Design
- `xor_gate_tb.v` – Testbench

## Key Concepts

- Dataflow Modelling
- Continuous Assignment (`assign`)
- Boolean XOR Operation
- Functional Verification
