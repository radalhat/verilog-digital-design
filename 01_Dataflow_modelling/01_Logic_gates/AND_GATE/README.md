# AND Gate

## Description

2-input AND gate implemented using **Dataflow Modelling** in Verilog HDL.

## Files

- `and_gate.v` – Design
- `and_gate_tb.v` – Testbench

## Key Concepts

- Dataflow Modelling
- Continuous Assignment (`assign`)
- Boolean AND Operation
- Functional Verification

## Output

`Y = A & B`
