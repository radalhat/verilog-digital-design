# XNOR Gate

## Description

2-input XNOR gate implemented using **Dataflow Modelling** in Verilog HDL.

## Files

- `xnor_gate.v` – Design
- `xnor_gate_tb.v` – Testbench

## Key Concepts

- Dataflow Modelling
- Continuous Assignment (`assign`)
- Boolean XNOR Operation
- Functional Verification

## Output

`Y = ~(A ^ B)`
