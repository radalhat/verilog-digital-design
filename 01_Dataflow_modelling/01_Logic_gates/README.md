# Logic Gates

## Overview

This folder contains the implementation of fundamental logic gates using the **Dataflow Modelling** style in Verilog HDL.

Each project includes the Verilog design, a testbench for functional verification, and supporting documentation.

## Implemented Gates

- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate

## Folder Structure

Each logic gate folder contains:

- Verilog HDL source code (`.v`)
- Testbench (`*_tb.v`)
- `README.md`
- Logic diagram (where applicable)
- Simulation waveform (where applicable)

## Tools Used

- Verilog HDL
- Synopsys VCS
- QuestaSim
- Git
- GitHub

## Learning Objectives

- Understand the operation of basic logic gates.
- Implement logic gates using Dataflow Modelling.
- Develop testbenches for verification.
- Simulate and verify digital circuits.
