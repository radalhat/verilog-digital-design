# Comparator

## Overview

A comparator is a combinational circuit that compares two binary numbers and determines whether one value is greater than, less than, or equal to the other.

This project implements comparators using **Verilog HDL** with the **Gate-Level Modelling** technique by connecting built-in Verilog gate primitives.

## Implemented Circuits

- 1-Bit Comparator
- 2-Bit Comparator
- 4-Bit Comparator

## Features

- Gate-Level Modelling
- Built using Verilog gate primitives
- Separate testbench for each comparator
- Verified using `$monitor`
- Clean and easy-to-understand implementation
- Suitable for learning digital comparison circuits

## Simulation Tool

- Synopsys Verdi

## Language

- Verilog HDL

## Learning Outcomes

After completing this project, you will be able to:

- Understand the operation of digital comparators.
- Design 1-bit, 2-bit, and 4-bit comparators using gate-level primitives.
- Implement combinational comparison circuits in Verilog HDL.
- Develop and simulate Verilog testbenches.
- Apply comparators in arithmetic and digital decision-making systems.
